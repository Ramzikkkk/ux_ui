# Мобильное приложение для записи в автосервис (Учебный проект)

Интерактивный кликабельный прототип мобильного приложения для записи на ремонт и техническое обслуживание автомобиля. 
Проект разработан в соответствии с требованиями ТЗ и методическими указаниями по UI/UX дизайну.

**Стек:** React, Tailwind CSS, Vite, React Router, Lucide-react.

---


## Запуск прототипа

Для запуска интерактивного прототипа на локальной машине:

### Шаг 1. Переход в папку проекта
```bash
cd prototype
```

### Шаг 2. Установка зависимостей
```bash
npm install
```

### Шаг 3. Запуск сервера разработки
```bash
npm run dev
```

После запуска в терминале появится ссылка (обычно `http://localhost:5173`). Откройте её в браузере. 
**Совет:** Для лучшего эффекта откройте инструменты разработчика в браузере (F12) и переключите режим отображения на "Мобильное устройство" (Responsive/Mobile view).

##  Разработка и дизайн

Проект создавался по методологии поэтапного проектирования интерфейсов:

1. **Research**: Анализ целевой аудитории и конкурентов $\rightarrow$ `research.md`
2. **UX Design**: Проектирование структуры и пользовательских потоков $\rightarrow$ `ux_structure.md`
3. **UI Design**: Создание визуального языка и библиотеки компонентов $\rightarrow$ `ui_design_system.md`
4. **Prototyping**: Сборка кликабельного приложения на React.

Все изменения в дизайне фиксировались в документации перед тем, как попасть в код, чтобы обеспечить согласованность интерфейса.
