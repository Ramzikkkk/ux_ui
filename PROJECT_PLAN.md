# План реализации учебного проекта: Мобильное приложение для записи в автосервис

## 🎯 Цель проекта
Создать качественный учебный кликабельный прототип мобильного приложения для записи на ремонт и ТО автомобиля, который полностью соответствует требованиям ТЗ и методических указаний.

**Принцип:** "Минимально необходимая реализация для получения высокой оценки".

---

## 🛠️ Стек технологий
Для создания высококачественного интерактивного прототипа выбран следующий стек:
- **Framework:** React (компонентный подход для быстрой разработки).
- **Styling:** Tailwind CSS (современный UI, высокая скорость верстки).
- **Build Tool:** Vite (быстрая сборка и запуск).
- **Routing:** React Router (навигация между всеми экранами приложения).
